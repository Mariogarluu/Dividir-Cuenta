# Dividir Cuenta

¡Bienvenido a **Dividir Cuenta**!  
Esta aplicación, desarrollada en **Kotlin**, facilita el proceso de dividir cuentas entre varias personas de manera rápida, sencilla y eficiente.

## 🚀 Características

- Divide el total de una cuenta entre varios participantes
- Añade o elimina personas fácilmente
- Calcula el monto individual automáticamente
- Interfaz intuitiva y fácil de usar
- 100% desarrollada en Kotlin

## 🛠️ Tecnologías utilizadas

- **Lenguaje principal:** Kotlin
- **Plataforma:** Android

## 📲 Instalación y uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Mariogarluu/Dividir-Cuenta.git
   ```
2. Abre el proyecto con Android Studio.
3. Compila y ejecuta la aplicación en tu dispositivo o emulador Android.
